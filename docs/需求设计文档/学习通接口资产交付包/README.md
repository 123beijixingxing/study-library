# 学习通接口资产交付包

## 1. 交付包说明

- 交付包版本：V1.1
- 编写日期：2026-04-24
- 交付目标：将学习通接口相关规范、联调、测试和代码样板整理为统一的正式交付目录，方便产品、前端、后端、测试、实施统一取用。

## 2. 目录结构

```text
学习通接口资产交付包/
  01-接口规范/
  02-联调导入/
  03-代码模板/
  04-测试资产/
  05-使用说明/
  asset-manifest.json
  README.md
```

## 3. 使用原则

1. `学习通OpenAPI草案.yaml` 作为接口结构主源。
2. Postman/Apifox 导入材料与 OpenAPI 草案保持同步更新。
3. `代码模板示例/` 用于研发快速建骨架，不作为生产代码直接上线。
4. 测试资产优先使用 CSV 导入版和 P1 冒烟版开展联调验证。

## 4. 推荐阅读顺序

1. 先看 `05-使用说明/操作手册.md`
2. 再按角色查看对应目录
3. 如需追踪源文件位置，查看 `asset-manifest.json`

## 5. 对应源文件

本交付包引用的核心源文件均位于 `docs/需求设计文档/`，清单详见 `asset-manifest.json`。
