## 3. 模块级验收清单

### 3.1 登录与鉴权

- [ ] 登录成功返回 `token`、`refreshToken`、`userInfo`
- [ ] `GET /auth/me` 能正确返回当前管理员信息
- [ ] 退出登录后 session 清空
- [ ] refreshToken 过期时能正确跳回登录页

### 3.2 用户模块

- [ ] 用户列表分页结构正确
- [ ] 用户筛选条件能生效
- [ ] 用户详情页字段与列表一致
- [ ] 权限配置保存后重新进入仍正确

### 3.3 课程与题库模块

- [ ] 课程列表筛选与状态切换可用
- [ ] 课程详情页章节数据正确
- [ ] 题库列表数量字段正确
- [ ] 题目详情中的选项、答案、解析保存正确
- [ ] 版本审计总台可展示真实版本链路

### 3.4 园地与审核模块

- [ ] 园地列表可按内容类型与状态筛选
- [ ] 火热审核状态切换正确
- [ ] 分享审核状态切换正确
- [ ] 举报处理结果与备注回显正确

### 3.5 会员、消息、客服、系统模块

- [ ] 会员列表、套餐、成长规则字段回显正确
- [ ] 消息管理和模板管理字段一致
- [ ] 客服列表与回复动作可用
- [ ] 系统配置保存后重新进入回显正确

### 3.6 分析与审计模块

- [ ] 统计页总量、趋势数据正常
- [ ] 日志页列表、筛选、时间格式正常
- [ ] 试卷分析页筛选、趋势、快照导出正确
- [ ] 版本审计总台筛选、导出、时间轴可用

## 4. 联调完成判定标准

可判定联调完成的最低标准：

1. 登录与 token 刷新链路通过
2. 用户、课程、题库、试卷四大主数据链路通过
3. 审核运营链路通过
4. 统计与日志链路通过
5. `yarn type-check` 通过
6. `yarn build` 通过
