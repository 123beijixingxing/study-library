# 学习通后台交付与验收清单

## 1. 文档说明

- 项目名称：学习通后台管理系统
- 对应前端工程：`study-library-admin`
- 文档目的：作为当前后台原型工程的交付确认、启动说明、演示路径和验收参考。

## 2. 当前交付状态

当前 `study-library-admin` 已完成一套可运行、可演示、可扩展的后台管理前端工程，具备以下特点：

- 页面层：后台主菜单页面与核心二级页面已落地。
- 数据层：支持本地 mock 持久化演示数据。
- 技术层：支持 `yarn dev`、`yarn type-check`、`yarn build`。
- 交互层：支持表格、筛选、弹窗、抽屉、详情页、审计页、分析资产管理页。
- 文档层：已补齐接口对接清单、子项目规范和交付验收清单。

结论：当前后台功能建设阶段已完成，可进入后端联调、测试验收或工程优化阶段。

## 3. 启动与验证命令

在 `study-library-admin` 目录下执行：

```bash
yarn
yarn dev
yarn type-check
yarn build
```

当前验证结果：

- `yarn`：通过
- `yarn dev`：可启动，默认地址 `http://localhost:10001/`
- `yarn type-check`：通过
- `yarn build`：通过

## 4. 登录与 mock 说明

- 默认 mock 登录账号：`admin / 123456`
- mock 开关：`.env.development` 中 `VITE_USE_MOCK=true`
- mock 持久化 key：`study-library-admin-mock-db`
- 如需重置演示数据：清除浏览器 `localStorage` 中的 `study-library-admin-mock-db`

## 6. 重点功能验收项

### 6.1 用户与权限

- 用户列表可正常筛选、分页、编辑、启停
- 用户详情可查看基础资料和统计摘要
- 用户权限配置页可修改角色与权限并保存

### 6.2 课程与题库

- 课程列表支持筛选、编辑、状态切换
- 课程详情管理支持章节增改
- 题库管理可进入题目列表
- 题目管理支持新增、导入、状态切换
- 题目详情页支持模板套用、选项编辑、答案与解析维护

### 6.3 版本审计

- 题目详情页支持复制为新版本
- 题目详情页支持版本差异对比与版本链导出
- 版本审计总台支持来源模式、版本范围、导出模块、导出分组等筛选
- 版本审计总台支持导出记录预览、重命名、再次导出、删除

### 6.4 园地与审核

- 园地管理支持火热/分享切换
- 火热内容审核、分享内容审核、举报审核可完成状态处理

### 6.5 会员、消息、客服

- 会员管理、套餐、成长规则页面可完成基本维护
- 消息中心与消息模板可维护消息资产
- 客服页可查看咨询并模拟回复

### 6.6 试卷分析资产库

- 试卷明细页支持章节筛选、课程对比、错题分析
- 支持保存筛选方案、方案重命名、方案收藏、方案标签编辑
- 支持导出全量快照/章节分析/试卷对比/错题分析
- 支持导出记录搜索、排序、分组、重点标记、预览、回放和复用为方案

## 7. 推荐演示路径

建议按如下顺序演示：

1. 登录后台 -> 仪表盘
2. 用户管理 -> 用户详情 -> 权限配置
3. 课程管理 -> 课程详情管理 -> 课程评价管理
4. 题库管理 -> 题目管理 -> 题目详情配置 -> 版本审计总台
5. 练习试卷 -> 试卷明细与分析 -> 筛选方案/快照导出
6. 园地管理 -> 火热审核 / 分享审核 / 举报审核
7. 会员管理 -> 套餐配置 / 成长规则 / 学习记录
8. 消息管理 -> 消息模板
9. 系统管理 -> 数据统计 -> 日志管理
