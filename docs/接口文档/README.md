# 接口文档目录

## 说明

- 本目录用于存放可直接对接、联调和交付的正式接口文档。
- 需求阶段草稿、接口设计过程文档仍保留在 `docs/需求设计文档/`。

## 推荐阅读顺序

1. `docs/接口文档/学习通前后台接口实现状态说明.md`
2. `docs/接口文档/学习通后台管理接口文档.md`
3. `docs/接口文档/学习通前台接口文档.md`
4. 如需按模块阅读，再进入对应拆分版目录

## 建议维护规则

1. 目录下优先放正式交付版本接口文档。
2. 若后续补前台接口文档，建议独立命名为 `学习通前台接口文档.md`。
3. 若接口变化较大，先更新 `docs/需求设计文档/` 中的设计文档，再同步到本目录。
