### 3.4 课程详情、练习与学习记录
#### 3.4.1 获取课程详情

- **Method**: GET
- **Uri**: `/api/app/v1/courses/{courseId}`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | courseId | long | 课程 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`courseId`、`courseName`、`teacherName`、`summary`、`tagList`、`averageScore`、`evaluationCount`、`progressPercent`、`chapterList`。

#### 3.4.2 获取课程章节列表

- **Method**: GET
- **Uri**: `/api/app/v1/courses/{courseId}/chapters`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | courseId | long | 课程 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`chapterId`、`chapterTitle`、`chapterType`、`sortNo`、`durationSeconds`、`learned`。

#### 3.4.3 获取课程评价列表

- **Method**: GET
- **Uri**: `/api/app/v1/courses/{courseId}/evaluations`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | courseId | long | 课程 ID | Y | 路径参数 |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 分页评价结构，关键字段：`evaluationId`、`score`、`content`、`userName`、`createdAt`。

#### 3.4.4 提交课程评价

- **Method**: POST
- **Uri**: `/api/app/v1/courses/{courseId}/evaluations`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | courseId | long | 课程 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**:

  ```json
  {
    "score": 5,
    "content": "课程讲解清晰，适合入门。"
  }
  ```

- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "evaluationId": 40001,
      "createdAt": "2026-05-12 10:00:00"
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.4.7 获取练习试卷列表

- **Method**: GET
- **Uri**: `/api/app/v1/practice/papers`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | scene | string | 场景 | N | |

  | chapterId | long | 章节 ID | N | |
  | paperType | string | 试卷类型 | N | `chapter` / `special` / `mock` |
  | pageNum | int | 页码 | N | 默认 `1` |

#### 3.4.9 提交练习试卷

- **Method**: POST
- **Uri**: `/api/app/v1/practice/papers/{paperId}/submit`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | paperId | long | 试卷 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**:

  ```json
  {
    "questionAnswerList": [
      {
        "questionId": 50001,
        "userAnswer": "B"
      }
    ],
    "costSeconds": 480
  }
  ```

- **Response**: 关键字段：`recordId`、`score`、`correctCount`、`wrongCount`。
