### 3.5 园地与互动
#### 3.5.1 获取园地内容列表

- **Method**: GET
- **Uri**: `/api/app/v1/garden/contents`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | tabType | string | 内容流类型 | N | `hot` / `share` / `favorite` |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 分页内容流结构，关键字段：`contentId`、`sourceName`、`contentText`、`likeCount`、`commentCount`、`favoriteCount`。

#### 3.5.2 获取内容详情

- **Method**: GET
- **Uri**: `/api/app/v1/garden/contents/{contentId}`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | contentId | long | 内容 ID | Y | 路径参数 |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 关键字段：`contentId`、`sourceInfo`、`contentText`、`imageList`、`liked`、`favorited`。

#### 3.5.3 关注内容来源

- **Method**: POST
- **Uri**: `/api/app/v1/garden/sources/{sourceId}/follow`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | sourceId | long | 来源 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`followed`。

#### 3.5.4 点赞内容

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | contentId | long | 内容 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`

#### 3.5.5 收藏内容

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | contentId | long | 内容 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`

#### 3.5.6 获取评论列表

- **Method**: GET
- **Uri**: `/api/app/v1/garden/contents/{contentId}/comments`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | contentId | long | 内容 ID | Y | 路径参数 |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 分页评论结构，关键字段：`commentId`、`nickname`、`commentText`、`createdAt`。

#### 3.5.7 发布评论

- **Method**: POST
- **Uri**: `/api/app/v1/garden/contents/{contentId}/comments`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | contentId | long | 内容 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**:

  ```json
  {
    "commentText": "内容很有帮助，感谢分享。"
  }
  ```

- **Response**: 关键字段：`commentId`、`createdAt`。

#### 3.5.9 举报内容

- **Method**: POST

- **Param**:

  | --- | ----- | ---- | -------- | ---- |

#### 3.5.10 提交联系留言

- **Method**: POST
- **Uri**: `/api/app/v1/garden/contact-messages`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`

  ```json

    "messageContent": "请尽快联系我处理课程问题。",
    "contactType": "phone"
  }
  ```

- **Response**: 关键字段：`messageId`。
