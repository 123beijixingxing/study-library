### 3.6 我的页与个人中心
#### 3.6.1 获取个人中心首页信息

- **Method**: GET
- **Uri**: `/api/app/v1/me/overview`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`username`、`avatar`、`memberInfo`。

#### 3.6.2 获取系统信息

- **Method**: GET
- **Uri**: `/api/app/v1/system/info`
- **Param**: NONE
- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 关键字段：`systemName`、`version`、`copyright`。

#### 3.6.3 获取扫一扫配置

- **Method**: GET
- **Uri**: `/api/app/v1/system/scan-config`
- **Param**: NONE
- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 关键字段：`enabled`、`scanType`、`scanTips`。

#### 3.6.4 获取会员中心信息

- **Method**: GET
- **Uri**: `/api/app/v1/me/member-center`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`memberLevel`、`expireTime`、`benefitList`。

#### 3.6.5 获取成长信息

- **Method**: GET
- **Uri**: `/api/app/v1/me/growth`
- **Param**: NONE

#### 3.6.6 获取签到状态

- **Method**: GET

- **Header**: `Authorization: Bearer <token>`

#### 3.6.7 用户签到

- **Param**: NONE

- **RequestBody**: NONE
- **Response**: 关键字段：`rewardInfo`、`checkedInAt`。

#### 3.6.8 获取任务列表

- **Method**: GET
- **Uri**: `/api/app/v1/me/tasks`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | taskType | string | 任务类型 | N | `check_in` / `study` / `practice` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`taskId`、`taskName`、`taskStatus`、`progressValue`。

#### 3.6.9 获取个人资料

- **Method**: GET
- **Uri**: `/api/app/v1/me/profile`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`username`、`nickname`、`email`、`phone`、`avatar`。

#### 3.6.10 更新个人资料

- **Method**: PUT
- **Uri**: `/api/app/v1/me/profile`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**:

  ```json
  {
    "nickname": "学习者小李",
    "email": "user01@example.com",
    "phone": "13800000001",
    "avatar": "https://cdn.example.com/avatar-new.png"
  }
  ```

- **Response**: 关键字段：`updatedAt`。

#### 3.6.11 获取离线下载列表

- **Method**: GET
- **Uri**: `/api/app/v1/me/offline-downloads`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`downloadId`、`courseName`、`downloadStatus`。

#### 3.6.12 删除离线下载

- **Method**: DELETE
- **Uri**: `/api/app/v1/me/offline-downloads/{downloadId}`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | downloadId | long | 下载记录 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 同 3.1.2。

#### 3.6.13 获取我的收藏

- **Method**: GET
- **Uri**: `/api/app/v1/me/favorites`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | favoriteType | string | 收藏类型 | N | `course` / `content` |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`favoriteId`、`targetId`、`title`。

#### 3.6.14 获取我的消息

- **Method**: GET
- **Uri**: `/api/app/v1/me/messages`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | messageStatus | string | 消息状态 | N | `unread` / `read` |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**: 关键字段：`messageId`、`title`、`messageType`、`sendTime`、`readStatus`。

#### 3.6.19 获取客服消息列表

- **Method**: GET
- **Uri**: `/api/app/v1/me/customer-service/sessions/{sessionId}/messages`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |

- **Header**: `Authorization: Bearer <token>`

- **Response**: 关键字段：`messageId`、`senderType`、`messageText`、`sendTime`。

#### 3.6.20 发送客服消息

- **Method**: POST
- **Uri**: `/api/app/v1/me/customer-service/sessions/{sessionId}/messages`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | sessionId | long | 会话 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**:

  ```json
  {
    "messageText": "您好，我的课程无法继续播放。"
  }
  ```

- **Response**: 关键字段：`messageId`、`sendTime`。
