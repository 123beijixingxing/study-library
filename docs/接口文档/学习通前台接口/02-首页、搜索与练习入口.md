| 序号 | 接口名称 | Method | URI |
| --- | --- | --- | --- |
| 1 | 获取首页热推内容 | GET | `/api/app/v1/home/hot-contents` |
| 2 | 获取首页课程列表 | GET | `/api/app/v1/home/courses` |

### 3.2 首页、搜索与练习入口
#### 3.2.1 获取首页热推内容

- **Method**: GET
- **Uri**: `/api/app/v1/home/hot-contents`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | limit | int | 热推条数 | N | 默认 `5` |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": [
      {
        "contentId": 3001,
        "title": "今日热门学习内容",
        "cover": "https://cdn.example.com/hot.png",
        "summary": "热门推荐摘要"
      }
    ],
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.2.2 获取首页课程列表

- **Method**: GET
- **Uri**: `/api/app/v1/home/courses`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | sectionType | string | 模块类型 | Y | `recommend` / `hot` / `latest` |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 分页课程卡片结构，关键字段：`courseId`、`courseName`、`summary`、`coverUrl`、`hotScore`、`badge`。

#### 3.2.3 获取首页课程分类列表

- **Method**: GET
- **Uri**: `/api/app/v1/home/categories`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 分页分类卡片结构，关键字段：`categoryId`、`categoryName`、`summary`、`hotScore`。

#### 3.2.4 获取搜索结果

- **Method**: GET
- **Uri**: `/api/app/v1/search`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | keyword | string | 搜索关键字 | Y | |
  | searchType | string | 搜索类型 | Y | `course` / `category` / `content` |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 分页搜索结果结构，关键字段：`resultType`、`resultId`、`title`、`summary`、`coverUrl`、`hotScore`。

#### 3.2.5 获取热门搜索词

- **Method**: GET
- **Uri**: `/api/app/v1/search/hot-keywords`
- **Param**:

  | scene | string | 场景 | N | `home` / `study` / `practice` |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 关键字段：`keywordList`。
