### 3.5 会员、会员套餐、成长规则、学习记录
权限点：`member:manage`

#### 3.5.1 查询会员列表

- **Method**: GET
- **Uri**: `/api/admin/v1/members`
- **Permission**: `member:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | keyword | string | 用户关键字 | N | 用户名模糊匹配 |
  | status | string | 会员状态 | N | `active` / `expiring` / `expired` / `cancelled` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 1,
      "list": [
        {
          "id": 10001,
          "username": "study_user_01",
          "memberLevel": "year_vip",
          "expireTime": "2027-04-24 23:59:59",
          "renewalCount": 3,
          "status": "active"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```
