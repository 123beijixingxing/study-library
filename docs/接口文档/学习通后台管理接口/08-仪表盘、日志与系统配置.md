### 3.8 仪表盘、日志与系统配置
权限点：`dashboard:view`、`logs:view`、`system:manage`

#### 3.8.1 查询仪表盘汇总

- **Method**: GET
- **Uri**: `/api/admin/v1/dashboard/summary`
- **Permission**: `dashboard:view`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "userTotal": 1280,
      "courseTotal": 86,
      "activeToday": 236,
      "memberTotal": 518
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.8.3 查询操作日志

- **Method**: GET
- **Uri**: `/api/admin/v1/logs/operations`
- **Permission**: `logs:view`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | keyword | string | 操作关键字 | N | |
  | operationType | string | 操作类型 | N | |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 1,
      "list": [
        {
          "id": 20001,
          "operateTime": "2026-05-11 11:10:00",
          "operatorName": "admin",
          "operationModule": "user",
          "operationType": "update",
          "operationContent": "更新用户状态",
          "ip": "127.0.0.1"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.8.4 查询登录日志

- **Method**: GET
- **Uri**: `/api/admin/v1/logs/logins`
- **Permission**: `logs:view`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | keyword | string | 用户关键字 | N | |
  | status | string | 登录状态 | N | `success` / `fail` / `logout` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 1,
      "list": [
        {
          "id": 21001,
          "loginTime": "2026-05-11 08:00:00",
          "username": "admin",
          "loginType": "password",
          "status": "success",
          "ip": "127.0.0.1"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```
