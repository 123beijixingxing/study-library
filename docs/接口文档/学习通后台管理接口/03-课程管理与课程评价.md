### 3.3 课程管理与课程评价
权限点：`course:manage`

#### 3.3.1 查询课程列表

- **Method**: GET
- **Uri**: `/api/admin/v1/courses`
- **Permission**: `course:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | categoryName | string | 分类名称 | N | |
  | status | string | 课程状态 | N | `draft` / `pending` / `published` / `offline` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 2,
      "list": [
        {
          "id": 2001,
          "courseName": "Python 入门基础",
          "categoryName": "编程基础",
          "hotScore": 98,
          "chapterCount": 8,
          "status": "published",
          "updateTime": "2026-05-11 10:00:00"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.3.7 查询课程评价列表

- **Method**: GET
- **Uri**: `/api/admin/v1/course-evaluations`
- **Permission**: `course:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | keyword | string | 评价关键字 | N | 用户名/内容模糊匹配 |
  | status | string | 评价状态 | N | `visible` / `hidden` / `deleted` |
  | courseName | string | 课程名称 | N | |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 1,
      "list": [
        {
  ```
