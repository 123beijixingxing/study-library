### 2.1 鉴权与权限

1. 除 `POST /api/admin/v1/auth/login` 与 `POST /api/admin/v1/auth/refresh` 外，其他接口均需要传入 `Authorization: Bearer <token>`。
2. 代码中通过 `@RequirePermission(...)` 控制菜单与操作权限，本文会在每个模块下标注对应权限点。
3. 登录与刷新接口支持可选请求头 `User-Agent`，用于记录会话环境。

### 2.3 通用查询约定

| key | value | 说明 | 是否必填 | 备注 |
| --- | ----- | ---- | -------- | ---- |

- **Method**: POST
- **Uri**: `/api/admin/v1/auth/login`
- **Permission**: NONE
- **Param**: NONE
