# 学习通前台接口文档 - 学习页与分类

> 来源：docs/接口文档/学习通前台接口文档.md

## 接口总览

| 序号 | 接口名称 | Method | URI |
| --- | --- | --- | --- |
| 1 | 获取课程分类树 | GET | `/api/app/v1/categories/tree` |
| 2 | 获取分类详情 | GET | `/api/app/v1/categories/{categoryId}` |
| 3 | 获取分类下课程列表 | GET | `/api/app/v1/categories/{categoryId}/courses` |
| 4 | 获取二级分类详情 | GET | `/api/app/v1/sub-categories/{subCategoryId}` |

### 3.3 学习页与分类
#### 3.3.1 获取课程分类树

- **Method**: GET
- **Uri**: `/api/app/v1/categories/tree`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | keyword | string | 分类关键字 | N | |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 分类树结构，关键字段：`categoryId`、`categoryName`、`children`。

#### 3.3.2 获取分类详情

- **Method**: GET
- **Uri**: `/api/app/v1/categories/{categoryId}`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | categoryId | long | 分类 ID | Y | 路径参数 |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 关键字段：`categoryId`、`categoryName`、`description`。

#### 3.3.3 获取分类下课程列表

- **Method**: GET
- **Uri**: `/api/app/v1/categories/{categoryId}/courses`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | categoryId | long | 分类 ID | Y | 路径参数 |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 分页课程卡片结构。

#### 3.3.4 获取二级分类详情

- **Method**: GET
- **Uri**: `/api/app/v1/sub-categories/{subCategoryId}`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | subCategoryId | long | 二级分类 ID | Y | 路径参数 |

- **Header**: NONE
- **RequestBody**: NONE
- **Response**: 关键字段：`subCategoryId`、`subCategoryName`、`courseList`。
