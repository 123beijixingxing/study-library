# 学习通前台接口文档 - 认证与账号

> 来源：docs/接口文档/学习通前台接口文档.md

## 接口总览

| 序号 | 接口名称 | Method | URI |
| --- | --- | --- | --- |
| 1 | 用户登录 | POST | `/api/app/v1/auth/login` |
| 2 | 用户退出登录 | POST | `/api/app/v1/auth/logout` |
| 3 | 用户注册 | POST | `/api/app/v1/auth/register` |
| 4 | 发送找回密码验证码 | POST | `/api/app/v1/auth/password/code` |
| 5 | 校验找回密码验证码 | POST | `/api/app/v1/auth/password/verify` |
| 6 | 重置密码 | POST | `/api/app/v1/auth/password/reset` |
| 7 | 获取当前用户信息 | GET | `/api/app/v1/auth/me` |

### 3.1 认证与账号
#### 3.1.1 用户登录

- **Method**: POST
- **Uri**: `/api/app/v1/auth/login`
- **Param**: NONE
- **Header**: NONE
- **RequestBody**:

  ```json
  {
    "username": "study_user_01",
    "password": "123456",
    "rememberMe": true
  }
  ```

- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "token": "string",
      "refreshToken": "string",
      "userInfo": {
        "userId": 10001,
        "username": "study_user_01",
        "avatar": "https://cdn.example.com/avatar.png",
        "memberStatus": "active",
        "roleList": ["app_user"]
      }
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.1.2 用户退出登录

- **Method**: POST
- **Uri**: `/api/app/v1/auth/logout`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "success": true
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.1.3 用户注册

- **Method**: POST
- **Uri**: `/api/app/v1/auth/register`
- **Param**: NONE
- **Header**: NONE
- **RequestBody**:

  ```json
  {
    "username": "study_user_02",
    "password": "123456",
    "confirmPassword": "123456",
    "email": "user02@example.com",
    "phone": "13800000002"
  }
  ```

- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "userId": 10002,
      "registerTime": "2026-05-12 09:00:00"
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.1.4 发送找回密码验证码

- **Method**: POST
- **Uri**: `/api/app/v1/auth/password/code`
- **Param**: NONE
- **Header**: NONE
- **RequestBody**:

  ```json
  {
    "accountType": "phone",
    "emailOrPhone": "13800000000"
  }
  ```

- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "expireSeconds": 300
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.1.5 校验找回密码验证码

- **Method**: POST
- **Uri**: `/api/app/v1/auth/password/verify`
- **Param**: NONE
- **Header**: NONE
- **RequestBody**:

  ```json
  {
    "emailOrPhone": "13800000000",
    "code": "123456"
  }
  ```

- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "verifyToken": "verify_token_xxx"
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.1.6 重置密码

- **Method**: POST
- **Uri**: `/api/app/v1/auth/password/reset`
- **Param**: NONE
- **Header**: NONE
- **RequestBody**:

  ```json
  {
    "verifyToken": "verify_token_xxx",
    "newPassword": "654321",
    "confirmPassword": "654321"
  }
  ```

- **Response**: 同 3.1.2。

#### 3.1.7 获取当前用户信息

- **Method**: GET
- **Uri**: `/api/app/v1/auth/me`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "userId": 10001,
      "username": "study_user_01",
      "avatar": "https://cdn.example.com/avatar.png",
      "memberStatus": "active",
      "roleList": ["app_user"]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```
