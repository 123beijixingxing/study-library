### 3.2 用户管理
权限点：`user:manage`

#### 3.2.1 查询用户列表

- **Method**: GET
- **Uri**: `/api/admin/v1/users`
- **Permission**: `user:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | keyword | string | 用户名/邮箱/手机号关键字 | N | 模糊匹配 |
  | role | string | 角色名称 | N | 如 `普通用户`、`VIP用户`、`管理员` |
  | status | string | 用户状态 | N | `enabled` / `disabled` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 2,
      "list": [
        {
          "id": 10001,
          "username": "study_user_01",
          "email": "user01@example.com",
          "phone": "13800000001",
          "role": "普通用户",
          "status": "enabled",
          "registerTime": "2026-04-24 10:00:00"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```
