### 3.6 园地内容与风控治理
权限点：`garden:manage`

#### 3.6.1 查询园地内容审核列表

- **Method**: GET
- **Uri**: `/api/admin/v1/contents`
- **Permission**: `garden:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | contentType | string | 内容类型 | N | 如 `hot` / `share` |
  | auditStatus | string | 审核状态 | N | `pending` / `approved` / `rejected` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 1,
      "list": [
        {
          "id": 14001,
          "contentType": "hot",
          "sourceName": "学习广场",
          "authorName": "study_user_01",
          "contentText": "这是一条园地内容摘要",
          "likeCount": 23,
          "commentCount": 6,
          "auditStatus": "pending",
          "publishTime": "2026-05-11 08:00:00"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.6.2 审核园地内容

- **Method**: POST
- **Uri**: `/api/admin/v1/contents/{contentId}/audit`
- **Permission**: `garden:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | contentId | long | 内容 ID | Y | 路径参数 |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**:

  ```json
  {
    "auditStatus": "approved"
  }
  ```

- **Response**: 同 3.6.1 列表项结构。
