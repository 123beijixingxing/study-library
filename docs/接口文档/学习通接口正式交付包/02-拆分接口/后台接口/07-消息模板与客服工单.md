### 3.7 消息模板与客服工单
权限点：`message:manage`、`service:manage`

#### 3.7.1 查询消息列表

- **Method**: GET
- **Uri**: `/api/admin/v1/messages`
- **Permission**: `message:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | keyword | string | 消息标题关键字 | N | |
  | sendStatus | string | 发送状态 | N | `draft` / `scheduled` / `sent` / `failed` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 1,
      "list": [
        {
          "id": 17001,
          "title": "课程更新提醒",
          "messageType": "system",
          "receiverType": "user",
          "sendStatus": "sent",
          "sendTime": "2026-05-11 12:00:00"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.7.2 保存消息任务

- **Method**: POST
- **Uri**: `/api/admin/v1/messages`
- **Permission**: `message:manage`
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**:

  ```json

    "id": 17001,
    "title": "课程更新提醒",
    "messageType": "system",
  ```

#### 3.7.3 查询消息模板列表

- **Method**: GET
- **Uri**: `/api/admin/v1/message-templates`
- **Permission**: `message:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | messageType | string | 消息类型 | N | `system` / `activity` / `service` |
  | status | string | 模板状态 | N | `draft` / `enabled` / `disabled` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 1,
      "list": [
        {
          "templateId": 18001,
          "templateName": "会员到期提醒模板",
          "messageType": "service",
          "titleTemplate": "您的会员将于 {expireDate} 到期",
          "contentTemplate": "尊敬的 {userName}，您的会员将于 {expireDate} 到期。",
          "channelList": ["site_message", "sms"],
          "status": "enabled",
          "updateTime": "2026-05-11 10:30:00"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```
