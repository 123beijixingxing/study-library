# 学习通后台管理接口文档 - 认证与登录

> 来源：docs/接口文档/学习通后台管理接口文档.md

## 接口总览

| 序号 | 接口名称 | Method | URI |
| --- | --- | --- | --- |
| 1 | 管理员登录 | POST | `/api/admin/v1/auth/login` |
| 2 | 刷新登录令牌 | POST | `/api/admin/v1/auth/refresh` |
| 3 | 退出登录 | POST | `/api/admin/v1/auth/logout` |
| 4 | 获取当前管理员信息 | GET | `/api/admin/v1/auth/me` |

### 3.1 认证与登录
#### 3.1.1 管理员登录

- **Method**: POST
- **Uri**: `/api/admin/v1/auth/login`
- **Permission**: NONE
- **Param**: NONE
- **Header**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | User-Agent | string | 客户端标识 | N | 用于会话记录 |

- **RequestBody**:

  ```json
  {
    "username": "admin",
    "password": "123456"
  }
  ```

- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "token": "string",
      "refreshToken": "string",
      "expiresIn": 7200,
      "refreshExpiresIn": 604800,
      "userInfo": {
        "userId": 1,
        "username": "admin",
        "roleList": ["管理员"]
      }
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

- **Uri**: `/api/admin/v1/auth/refresh`
- **Permission**: NONE
- **Param**: NONE
- **Header**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | User-Agent | string | 客户端标识 | N | 用于刷新会话 |

- **RequestBody**:

  ```json
  {
    "refreshToken": "string"
  }
  ```

- **Response**:

  ```json
    "code": 0,
    "message": "success",
    "data": {
      "token": "string",
      "refreshToken": "string",
      "expiresIn": 7200,
      "refreshExpiresIn": 604800,
      "userInfo": {
        "userId": 1,
        "username": "admin",
        "roleList": ["管理员"]
      }
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

#### 3.1.3 退出登录

- **Method**: POST
- **Uri**: `/api/admin/v1/auth/logout`
- **Permission**: 已登录即可
- **Param**: NONE
- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "success": true
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```

- **Uri**: `/api/admin/v1/auth/me`
- **Permission**: 已登录即可

- **RequestBody**: NONE
- **Response**:

  ```json
      "userId": 1,
      "username": "admin",
      "roleList": ["管理员"]
  ```
