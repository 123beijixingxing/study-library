### 3.4 首页配置、搜索运营、推荐位、题库与练习
权限点：`course:manage`

#### 3.4.1 查询首页课程配置列表

- **Method**: GET
- **Uri**: `/api/admin/v1/home/courses`
- **Permission**: `course:manage`
- **Param**:

  | key | value | 说明 | 是否必填 | 备注 |
  | --- | ----- | ---- | -------- | ---- |
  | pageNum | int | 页码 | N | 默认 `1` |
  | pageSize | int | 每页条数 | N | 默认 `10` |
  | sectionType | string | 模块类型 | N | `recommend` / `hot` / `latest` |

- **Header**: `Authorization: Bearer <token>`
- **RequestBody**: NONE
- **Response**:

  ```json
  {
    "code": 0,
    "message": "success",
    "data": {
      "pageNum": 1,
      "pageSize": 10,
      "total": 1,
      "list": [
        {
          "id": 5001,
          "sectionType": "recommend",
          "title": "首页推荐课程",
          "summary": "首页顶部推荐位课程",
          "hotScore": 98,
          "sortNo": 1,
          "displayStatus": "visible"
        }
      ]
    },
    "requestId": "uuid",
    "timestamp": 1715400000000
  }
  ```
