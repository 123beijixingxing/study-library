# 学习通后台权限点索引

## 1.说明

- 对应后端工程：`study-library-backend`
- 对应总文档：`docs/接口文档/学习通后台管理接口文档.md`
- 文档目的：汇总后台接口的权限点、对应模块和 URI 范围，方便菜单权限、按钮权限、角色分配和联调排查。

## 2.权限点总览

| 权限点 | 模块 | 主要 URI 范围 | 说明 |
| --- | --- | --- | --- |
| `user:manage` | 用户管理 | `/api/admin/v1/users*` | 用户增删改查、状态管理、权限配置 |
| `course:manage` | 课程与运营配置 | `/api/admin/v1/courses*`、`/api/admin/v1/course-evaluations*`、`/api/admin/v1/home/courses*`、`/api/admin/v1/question-banks*`、`/api/admin/v1/practice-papers*`、`/api/admin/v1/search/hot-keywords*`、`/api/admin/v1/recommend-slots*` | 课程、首页配置、题库、试卷、搜索运营、推荐位 |
| `member:manage` | 会员与成长 | `/api/admin/v1/members*`、`/api/admin/v1/member-packages*`、`/api/admin/v1/growth-rules*`、`/api/admin/v1/study-records*` | 会员、套餐、成长规则、学习记录 |
| `garden:manage` | 园地与风控 | `/api/admin/v1/contents*`、`/api/admin/v1/reports*`、`/api/admin/v1/sensitive-words*` | 内容审核、举报处理、敏感词 |
| `message:manage` | 消息中心 | `/api/admin/v1/messages*`、`/api/admin/v1/message-templates*` | 消息任务、消息模板 |
| `service:manage` | 客服工单 | `/api/admin/v1/service-tickets*` | 客服工单查询与回复 |
| `dashboard:view` | 仪表盘 | `/api/admin/v1/dashboard*` | 汇总统计、趋势图 |
| `logs:view` | 日志 | `/api/admin/v1/logs*` | 操作日志、登录日志 |
| `system:manage` | 系统配置 | `/api/admin/v1/system*` | 系统信息、功能开关 |

## 3. 无权限点或仅登录校验接口

| 接口 | 说明 |
| --- | --- |
| `POST /api/admin/v1/auth/login` | 登录接口，公开访问 |
| `POST /api/admin/v1/auth/refresh` | 刷新令牌接口，公开访问 |
| `POST /api/admin/v1/auth/logout` | 仅登录校验 |
| `GET /api/admin/v1/auth/me` | 仅登录校验 |

## 5. 使用建议

1. 若前端做菜单权限控制，可直接按本表建立模块到权限点映射。
2. 若后端新增接口，建议先确定归属权限点，再同步更新本文件与总接口文档。
3. 若后续扩展按钮级细分权限，可在本文件下追加“二级权限点”索引。
