# 学习通接口联调报文集

## 1. 文档说明

- 项目名称：学习通管理项目
- 文档版本：V1.1
- 编写日期：2026-04-24
- 对应文档：`docs/需求设计文档/学习通接口清单.md`
- 对应文档：`docs/需求设计文档/学习通接口DTO字段字典.md`
- 对应文档：`docs/需求设计文档/学习通接口成功响应示例集.md`
- 对应文档：`docs/需求设计文档/学习通接口错误响应示例集.md`
- 文档目的：提供前后台关键接口的请求头、请求参数、请求报文和响应报文示例，方便前后端联调和接口验收。

## 2. 联调约定

| 项 | 说明 |
| --- | --- |
| Base URL | 前台 `/api/app/v1`，后台 `/api/admin/v1` |
| Content-Type | 默认 `application/json` |
| 鉴权头 | `Authorization: Bearer <token>` |
| 时间格式 | 统一使用 `YYYY-MM-DD HH:mm:ss` 或毫秒时间戳 |
| 错误联调 | 失败报文优先参考 `docs/需求设计文档/学习通接口错误响应示例集.md` |

#### 请求

```http
POST /api/app/v1/auth/login HTTP/1.1
Content-Type: application/json
```

## 5. 联调检查清单

| 检查项 | 说明 |
| --- | --- |
| 枚举值一致性 | 校验接口中的状态值是否与枚举文档一致 |
| 字段命名一致性 | 校验响应字段是否与 DTO 字典一致 |
| 状态前置条件 | 校验非法状态是否按约定报错 |
| 时间字段格式 | 校验时间字符串或时间戳格式统一 |
| 列表分页结构 | 校验分页字段完整返回 |
| `requestId` 回传 | 确认每次响应都带追踪 ID |
