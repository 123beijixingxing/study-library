# 学习通接口文档交付说明

## 1. 交付说明

- 文档目录：`docs/接口文档`
- 交付目标：提供可直接用于前后端联调、测试验收、实施交接的正式接口文档集合
- 对应设计源：`docs/需求设计文档`

## 2. 交付范围

当前已交付内容包括：

| 类别 | 文件/目录 | 说明 |
| --- | --- | --- |
| 后台总文档 | `docs/接口文档/学习通后台管理接口文档.md` | 基于实际后端代码整理的后台接口明细 |
| 后台拆分版 | `docs/接口文档/学习通后台管理接口/` | 按模块拆分后的后台接口文档 |
| 后台权限索引 | `docs/接口文档/学习通后台权限点索引.md` | 后台权限点与 URI 范围映射 |
| 前台总文档 | `docs/接口文档/学习通前台接口文档.md` | 基于需求、接口清单与 OpenAPI 草案整理的前台接口文档 |
| 前台拆分版 | `docs/接口文档/学习通前台接口/` | 按模块拆分后的前台接口文档 |
| 实现状态说明 | `docs/接口文档/学习通前后台接口实现状态说明.md` | 说明前后台接口当前实现/设计状态 |
| 文档索引 | `docs/接口文档/README.md` | 本目录统一导航入口 |

## 3. 当前完成状态

### 3.1 后台接口

- 已按 `study-library-backend` 控制器实现进行回刷整理
- 已补模块总览、URI 索引、拆分版模块文档
- 已补权限点索引，适合角色配置和联调排查

### 3.2 前台接口

- 已按需求文档、接口清单和 OpenAPI 草案整理完成
- 当前仍属于“设计态接口文档”
- 待前台控制器代码落地后，可再次按实际实现回刷

## 4. 推荐使用方式

### 4.1 产品/项目经理

1. 查看 `docs/接口文档/学习通前后台接口实现状态说明.md`
2. 查看前台或后台总文档掌握接口范围

### 4.2 前端开发

1. 前台先看 `docs/接口文档/学习通前台接口文档.md`
2. 后台管理台看 `docs/接口文档/学习通后台管理接口文档.md`
3. 若按模块开发，进入对应拆分版目录

### 4.3 后端开发

1. 以 `study-library-backend` 实际代码为准
2. 同步参考 `docs/接口文档/学习通后台管理接口文档.md`
3. 需要权限校验时参考 `docs/接口文档/学习通后台权限点索引.md`

### 4.4 测试/实施

1. 先看 `docs/接口文档/README.md`
2. 再按前后台选择总文档或拆分版模块文档
3. 若需工具联调，参考 `docs/接口文档/学习通联调与工具资产索引.md`

## 5. 维护建议

1. 设计变更先更新 `docs/需求设计文档`。
2. 对外交付前再同步更新 `docs/接口文档`。
3. 后台新增真实接口时，优先更新总文档，再刷新拆分版。
4. 前台真实控制器落地后，应将“设计态接口文档”升级为“实现态接口文档”。

## 6. 完成结论

当前 `docs/接口文档` 已可作为正式接口文档目录使用，支持：

- 文档查阅
- 模块联调
- 权限排查
- 前后端交接
- 测试用例编写
