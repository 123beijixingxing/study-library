## 1. 说明

- 本文档用于索引接口联调、OpenAPI、Postman、Apifox 等工具资产
- 资产主要位于 `docs/需求设计文档`

## 2. OpenAPI 资产

| 文件 | 路径 | 说明 |
| --- | --- | --- |
| OpenAPI 草案 | `docs/需求设计文档/学习通OpenAPI草案.yaml` | 当前前后台统一接口结构草案，路径已包含完整 `/api/app/v1` 和 `/api/admin/v1` 前缀 |

### 使用建议

1. Swagger / Apifox 导入优先使用 `学习通OpenAPI草案.yaml`
2. 该草案已修正 `servers` 根路径，避免与完整 URI 重复拼接

## 6. 推荐联调顺序

1. 先阅读 `docs/接口文档/学习通前后台接口实现状态说明.md`
2. 再阅读前台或后台总接口文档
3. 然后导入 `学习通OpenAPI草案.yaml` 或 Postman 扩展集合
4. 最后结合示例报文集与状态码规范开展联调
